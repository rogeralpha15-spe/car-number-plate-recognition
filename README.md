# Car Number Plate Recognition

This is our FIT CW1 project for demonstrating a car number plate recognition system deployed on OpenStack.

## Features
- Car image input
- Number plate cropping
- OCR using Tesseract
- GitHub repository for version control
- GitHub Actions for CI/CD
- KPI comparison between OpenStack and local deployment

## Technologies
- Python
- OpenCV
- Tesseract OCR
- GitHub
- GitHub Actions
- OpenStack

## Demo Flow
1. Input image
2. Crop number plate
3. Preprocess image
4. Extract text using OCR
5. Show final plate text

## Note
The original experimentation was based on a Kaggle notebook. The final demo version was reorganized into Python files for deployment and presentation.
